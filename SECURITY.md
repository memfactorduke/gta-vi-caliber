# Security Policy

## Reporting a vulnerability

Please report security issues privately via
[GitHub Security Advisories](../../security/advisories/new) — do not open a
public issue for exploitable problems.

## Scope

- The game client (code in `game/` and `engine/`)
- CI workflows and build tooling in this repository

Engine vulnerabilities in Godot itself should be reported upstream to the
[Godot security policy](https://github.com/godotengine/godot/blob/master/SECURITY.md).

## Response

This is a volunteer project; we aim to acknowledge reports within 7 days.
