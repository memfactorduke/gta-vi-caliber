# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version:

- **Be welcoming.** Programmers, artists, writers, playtesters, first-timers,
  veterans, humans, and AI-assisted contributors are all here to build the
  same game.
- **Critique work, not people.** "This streaming approach stalls the main
  thread" is useful; "you clearly don't understand engines" is not.
- **No harassment** of any kind — in issues, PRs, commits, or community
  channels. No hate speech, no sexualized content, no doxxing.
- **Respect maintainer decisions.** Disagreement is fine and welcome in the
  issue thread; campaigns and pile-ons are not.

## Enforcement

Report violations privately to the maintainers at **duola@hypercho.com**.
Reports are handled confidentially. Consequences scale from a warning to a
permanent ban at maintainer discretion, per the Contributor Covenant's
enforcement guidelines.
